# Claude Usage Tracker — Chrome

## Installation (Mode développeur)

1. Téléchargez et décompressez le ZIP `claude-usage-chrome.zip`
2. Ouvrez Chrome → `chrome://extensions`
3. Activez le **Mode développeur** (switch en haut à droite)
4. Cliquez **"Charger l'extension non empaquetée"**
5. Sélectionnez le dossier `claude-usage-chrome/`
6. L'extension apparaît dans la barre ✓

## Compatibilité

- Chrome 88+ (Manifest V3)
- Edge, Brave, Arc et tout navigateur Chromium : même procédure

## Mise à jour

Re-déposez le dossier via "Charger l'extension non empaquetée" ou cliquez le bouton ↺ dans chrome://extensions.
